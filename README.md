# Fake News Detection System

A binary text classification model that distinguishes real news from fake news using NLP techniques and machine learning.

## 🎯 Problem Statement
Misinformation spreads rapidly online. This project builds an automated classifier to flag fake news articles, demonstrating practical AI application toward content verification and cybersecurity.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** scikit-learn
- **NLP Techniques:** Tokenization, TF-IDF Vectorization, Feature Selection

## ⚙️ How It Works
1. Raw news text is preprocessed using tokenization and cleaned for noise
2. TF-IDF vectorization converts text into numerical feature vectors
3. Feature selection reduces dimensionality, improving speed and accuracy
4. A binary classification model is trained to predict real vs. fake

## 📊 Results
- **>90% accuracy** on held-out test set
- **~30% reduction** in model inference time through optimized preprocessing pipeline
